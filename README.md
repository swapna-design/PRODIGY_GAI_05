# PRODIGY_GAI_05
Implemented Neural Style Transfer to generate artistic images by blending content and style from two different images
Overview

In this project, I implemented Neural Style Transfer to combine the content of one image with the artistic style of another image. Using a pre-trained deep learning model, I generated a stylized image while preserving the original content.

Objective

To apply the artistic style of one image to another image using Neural Style Transfer techniques.

Tools and Technologies
Python
Google Colab
TensorFlow
TensorFlow Hub
Matplotlib
Features
Content image processing
Style image processing
Neural Style Transfer implementation
Generation of stylized images
Outcome

Successfully generated artistic images by combining the content of one image with the style of another image using deep learning.
